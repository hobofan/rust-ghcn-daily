# Rust GHCN-Daily parser

Rust crate providing a parser and data structures for working with the GHCN-Daily dataset.

### Installation

To start using this crate add the following to your `Cargo.toml`:

```toml
[dependencies]
ghcn-daily = "0.0.1"
```
